Bootstrap Cards

A card in Bootstrap 5 is a bordered box with some padding around its content. It includes options for headers, footers, content, colors, etc.

Example

                
                    <div class="card" style="width:20rem;margin:20px 0 24px 0">

                        <img class="card-img-top" src="img_avatar3.png" alt="image" style="width:100%">

                        <div class="card-body">

                          <h4 class="card-title">John Doe</h4>

                          <p class="card-text">Some example text some example text. John Doe is an architect and engineer</p>

                          <a href="javascript:void(0)" class="btn btn-primary">See Profile</a>

                             </div>

                      </div>
