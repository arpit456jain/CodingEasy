What is Machine Learning?

  • Machine Learning is a way by which machines are made to think like humans and process tasks with the prior knowledge of some past datasets.
  • We generally use 3 terms related to this field - Artificial Intelligence (AI), Machine Learning (ML), Deep Learning (DL).
  • ML is a subset of AI. And DL is a subset of ML
  • If Artificial Intelligence is a subject, then Machine Learning is just a theory to solve various kinds of AI Problems. And the AI problems that are unsolvable by ML Algorithms can be solved by DL. This is because Deep Learning includes usage of Neural Networks

More About Machine Learning

Machine Learning includes a process called Training where dataset is fed to the machine.

We choose a particular Machine Learning Algorithm to proceed with the problem.

The algorithm gradually develops a relationship between features and corresponding labels
