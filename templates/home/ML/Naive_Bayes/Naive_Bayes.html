Decision Tree is a Classification Algorithm. This algorithm works on the mathematical concepts of Baye's Theorem which is Conditional Probability.

The dataset is converted to a frequency table and then Baye's Theorem is applied to calculated the predicted results. This algorithm gives good results when all features are categorical

The formula used is used where A is the target variable which needs to be predicted. And B is one value taken from each of the features. Suppose we have 4 features so the model asks questions to each feature and then determines the value of B for each feature. After this finally we get value of A.

There are total 3 types of Naive Bayes Classifier-

  1. Gaussian Naive Bayes
  2. Multinomial Naive Bayes
  3. Bernoulli Naive Bayes
