GridSearch


  • Grid search is the simplest algorithm for hyperparameter tuning. Basically, we divide the domain of the hyperparameters into a discrete grid. Then, we try every combination of values of this grid, calculating some performance metrics using cross-validation.


GridSearch

The process of choosing an optimal parameter is known as hyper parameter tuning. this ca be achieved using GridSearchCV.

It can run through all the different parameters that is fed into the parameter grid. Then, it produces the best combination of parameters. This done based on a scoring metric that we can give as per our choice like accuracy, f1, etc.

GridSearchCV has some cons. In GridSearchCV, the best parameters are limited and the process is time-consuming.

sklearn library provides the GridSearchCV function.

Below is given a sample code performed on the Iris dataset using GridSearchCV. from sklearn.model_selection import GridSearchCV clf = GridSearchCV(svm.SVC(gamma='auto'), { 'C': [1,10,20], 'kernel': ['rbf','linear'] }, cv=5, return_train_score=False) clf.fit(iris.data, iris.target) clf.cv_results_
