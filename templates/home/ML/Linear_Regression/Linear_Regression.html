Linear Regression (Regression Algorithm)



Linear Regression

Linear Regression is the most commonly used Regression technique which involves dealing with numerical values and some mathematical calculations

This algorithm uses multiple mathematical & statistical techniques like Linear Algebra, Probability, to find the final results

Linear Regression is widely used in predicting the prices of any object. Like we can use it for problem statements like House Price, Car Price, Stock Price etc

As shown in the plot the circle markers(Blue Dots) are actual values and the best fit line(Red Line) represents the predicted values.

After this we try to find the error which is the difference between predicted and actual value. We also try to minimize the error and thus adjust the best fit line so that we get minimum error. Once we get the perfect best fit line for our problem then we are all good to go for predicting unknown values.

Need For Regression Analysis

Regression analysis helps us to understand the relationship between two or more variables by creating a predictive model.

Y = f(X)

Where Y is Dependent Variable on X

X is the independent Variable on Y

Measure of Good Fit

We find the best fit line on the basis of the value of ∑[(Y0-Yp)^2]. Where Y0 is the actual value and Yp is the predicted value.

Types of Linear Regression

1. Simple Linear Regression

2. Multiple Linear Regression

Simple Linear Regression

Simple linear regression is used to model the relationship between two continuous variables.

Formula

Y = a + bX

Where Y = y-coordinate

Where X = x-coordinate

Where b = slope

Where a = y-intercept

Multiple Linear Regression

In this we have one dependent variables and more than one independent variable.

Formula

Y = b0 + b1X + b2X^2 + b3X^3 + ... + bnX^n + e

Y = Dependent variable

X = Independent Variable

b0 = y-intercept

bn = Slope coefficient for each explanatory variable

e = The model's error term
