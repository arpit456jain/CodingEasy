Random Forest (Regression & Classification Algorithm)



Random Forest

Random Forest Tree is a Regression & Classification Algorithm. When used for Regression it's called Random Forest Regressor & when it's used for Classification it's called Random Forest Classifier.

The concept is very similar to the decision tree algorithm. It works by taking multiple trees into consideration. And in the end it combines the results of all the trees taken into count.

Suppose we have 2 classes - Class 0 & Class 1. And we are considering 3 trees together. So the majority decision of all the 3 trees will be considered. Like in the given plot the two trees give output as Class 0 and one tree gives output as Class 1. Thus majority voting is for Class 0 and thus the result will be declared as Class 0. The data is divided into 3 sections here and fed to each of the trees.

When we combine more than 1 models in a problem statement then it is called Ensemble Learning. Since in Random Forest Algorithm we combine results of multiple decision trees so it is also called Ensemble Learning.
