What is C

C is a high-level and general-purpose programming language that is ideal for developing firmware or portable applications. Originally intended for writing system software, C was developed at Bell Labs by Dennis Ritchie for the Unix Operating System in the early 1970s. Ranked among the most widely used languages, C has a compiler for most computer systems and has influenced many popular languages â€“ notably C++



C Basic Syntax

#include <stdio.h> includes the standard input output library functions. The printf() function is defined in stdio.h .
int main() - The main() function is the entry point of every program in c language.
printf() - The printf() function is used to print data on the console.
return 0 - The return 0 statement, returns execution status to the OS. The 0 value is used for successful execution and 1 for unsuccessful execution.

Example

           
            #include <stdio.h> 

                int main() {
                    printf("Hello World!");
                    return 0;
